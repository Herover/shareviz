# Documentation

Terms used in the documentation

* **DataTortilla/DT**: the software you are reading the documentation for.
* **Editor**: a user who use DataTortilla for creating and sharing charts.
* **Viewer**: a user who consume charts, either through embedded web sites or pictures.
* **Sys admin**: a system administrator who takes care of the technical side of using DataTortilla.
* **Developer**: someone who can write code, typically by contributing to DataTortilla or creating custom plugins.

It's expected that some of these roles overlap a bit.

Keep in mind that DataTortilla is in a unstable state, meaning breaking changes will happen. Keep a eye out for this when you upgrade.
